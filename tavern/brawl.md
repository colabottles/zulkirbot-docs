# Tavern Brawl

A brawl can randomly break out when a player buys a drink or meal (15% chance).

## Joining

When a brawl is announced, type `!brawl` within **30 seconds** to join. You must have bought a drink or meal first.

## Rules

- Minimum **2 players** required.
- Free-for-all — everyone fights everyone.
- Last one standing wins.
- Losers drop to **0 HP** — use `!rest` to recover.

## Rewards

Rewards scale with the number of participants:

| Reward | Amount |
| --- | --- |
| Gold | 15g × number of fighters |
| XP | 10 XP × number of fighters |
