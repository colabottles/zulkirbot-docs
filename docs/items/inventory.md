# Inventory

| Command | Description |
| --- | --- |
| `!inventory` | View your items. |
| `!equip [item]` | Equip an item from your inventory. |
| `!unequip [slot]` | Unequip an item from a slot. |
| `!use [item]` | Use a scroll or potion. |
| `!potion` | Drink a potion from your inventory. |
| `!drop [item]` | Permanently discard an item. |

> [!WARNING]
> You cannot drop or sell equipped or cursed items.
