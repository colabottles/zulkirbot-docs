# The Tavern

| Command | Description |
| --- | --- |
| `!tavern` | Welcome message with available tavern commands. |
| `!drinks` | Browse the full drinks menu. |
| `!meals` | Browse the full meals menu. |
| `!tavern drink [name]` | Order a drink. Buffs last for your next fight. |
| `!tavern meal [name]` | Order a meal to restore lost HP. |
| `!barkeep` | Hear from the barkeep. Available after your first purchase. |
| `!rumour` | Hear a rumour from a fellow patron. Available after your first purchase. |
| `!gamble [amount] [number 0-36] [red/black]` | Spin the roulette wheel. Exact number pays 35:1, color match pays 1:1. 0 is green and beats everything. |
